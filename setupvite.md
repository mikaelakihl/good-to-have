# How to set up Vite project

## Github
- [ ] Create repo in Github
- [ ] >Git clone repo in VS code

## Install Vite and TypeScript

- [ ] Install npm in Terminal
 ```bash
  npm create vite@latest .
```
Choose Vanilla -> TypeScript
  
In terminal

```bash
  npm install
```

```bash
  npm run dev
```
